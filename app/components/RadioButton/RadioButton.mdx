# RadioButton

O componente **RadioButton** é usado para capturar seleções exclusivas dentro de um grupo.

## Propriedades (Props)

### `selected`
- **Tipo:** `boolean`
- **Descrição:** Define se o radio button está selecionado.
- **Padrão:** `false`

### `onSelect`
- **Tipo:** `() => void`
- **Descrição:** Callback chamado quando o radio button é selecionado.

### `label`
- **Tipo:** `string`
- **Descrição:** Texto opcional exibido ao lado do radio button.

### `name`
- **Tipo:** `string`
- **Descrição:** Nome do grupo de radio buttons, necessário para seleção exclusiva.

## Exemplo de Uso

```tsx
<RadioButton selected={true} label="Opção A" name="grupo1" onSelect={() => console.log("Selecionado")} />
```

## Variações

### RadioButton Não Selecionado
```tsx
<RadioButton selected={false} label="Opção 1" name="grupo1" />
```

### RadioButton Selecionado
```tsx
<RadioButton selected={true} label="Opção 2" name="grupo1" />
