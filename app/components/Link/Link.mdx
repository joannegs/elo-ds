# Link

O componente **Link** é utilizado para criar links navegáveis na interface do usuário. Ele suporta estados desabilitados e permite personalização por meio de propriedades e classes CSS.

----------

## Propriedades (Props)

O componente **Link** aceita as seguintes propriedades:

### `children`

-   **Tipo:** `React.ReactNode`
-   **Descrição:** O conteúdo exibido dentro do link, geralmente texto.

### `href`

-   **Tipo:** `string` (opcional)
-   **Descrição:** O endereço (URL) para onde o link aponta.
-   **Exemplo:** `"/home"` ou `"https://example.com"`.

### `disabled`

-   **Tipo:** `boolean` (opcional)
-   **Descrição:** Define se o link está desabilitado.
-   **Comportamento:**
    -   Quando `true`, o link não pode ser clicado e o atributo `href` é removido.
    -   A classe `aria-disabled` é adicionada para fins de acessibilidade.
-   **Padrão:** `false`.

### `underline`

-   **Tipo:** `boolean` (opcional)
-   **Descrição:** Aplica `hover:underline` ao link, exibindo sublinhado ao passar o mouse.
-   **Padrão:** `false`.

### `className`

-   **Tipo:** `string` (opcional)
-   **Descrição:** Permite adicionar classes CSS personalizadas para ajustar o estilo do link.

### Atributos HTML

-   O componente aceita todos os atributos padrão de um elemento `<a>`, incluindo:
    -   `target`: Define como o link será aberto (`_blank`, `_self`, etc.).
    -   `rel`: Especifica a relação do link com o documento atual.

----------

## Exemplo de Uso

### Link Navegável

```tsx
<Link href="https://example.com">Visite o site</Link>
```

### Link Desabilitado

```tsx
<Link href="https://example.com" disabled>
  Link Indisponível
</Link>
```

### Link com Sublinhado ao Hover

```tsx
<Link href="/dashboard" underline>
  Ir para o Dashboard
</Link>
```
