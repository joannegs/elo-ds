# Badge

O componente **Badge** é utilizado para exibir pequenas etiquetas de informação. Ele é estilizado e configurável para atender diferentes cenários e necessidades.

----------

## Propriedades (Props)

O componente **Badge** aceita as seguintes propriedades:

### `color`

-   **Tipo:** `string`
-   **Descrição:** Define a cor de fundo do badge.
-   **Exemplo:** `"primary"`, `"success"`, `"error"`

### `textColor`

-   **Tipo:** `string`
-   **Descrição:** Define a cor do texto do badge.
-   **Exemplo:** `"white"`, `"black"`, `"gray"`

### `tagContent`

-   **Tipo:** `string`
-   **Descrição:** O conteúdo do badge, geralmente texto.
-   **Exemplo:** `"Novo"`, `"Promoção"`

### `shadow`

-   **Tipo:** `boolean`
-   **Descrição:** Define se o badge terá sombra.
-   **Padrão:** `false`

### `size`

-   **Tipo:** `"sm" | "md" | "lg"`
-   **Descrição:** Define o tamanho do badge.
-   **Valores possíveis:**
    -   `sm`: Badge pequeno.
    -   `md`: Badge médio (padrão).
    -   `lg`: Badge grande.

### `iconStart`

-   **Tipo:** `InputIconProps` (opcional)
-   **Descrição:** Define o ícone a ser exibido no início do badge.
-   **Valores possíveis:** Um objeto com as propriedades:
    - `iconName`: Nome do ícone a ser usado (veja as opções de ícones disponíveis).
    - `solid`: Define se o ícone será sólido ou com contorno. **Padrão:** `false`.

### `iconEnd`

-   **Tipo:** `InputIconProps` (opcional)
-   **Descrição:** Define o ícone a ser exibido no final do badge.
-   **Valores possíveis:** Um objeto com as propriedades:
    - `iconName`: Nome do ícone a ser usado (veja as opções de ícones disponíveis).
    - `solid`: Define se o ícone será sólido ou com contorno. **Padrão:** `false`.

### `className`

-   **Tipo:** `string` (opcional)
-   **Descrição:** Permite adicionar classes CSS personalizadas para ajustar o estilo do badge.

### Atributos HTML

-   O badge aceita todas as propriedades padrão de um elemento `<div>` no HTML, incluindo:
    -   `id`: Identificador único.

----------

## Exemplo de Uso

### Badge Primário

```tsx
<Badge color="primary" textColor="white" tagContent="Novo" />`
```

### Badge de Sucesso

```tsx
<Badge color="success" textColor="white" tagContent="Sucesso" />
```
### Badge de Erro

```tsx
<Badge color="error" textColor="white" tagContent="Erro" />
```

### Badge com Sombra

```tsx
<Badge color="primary" textColor="white" shadow tagContent="Com Sombra" />
```

### Badge Grande

```tsx
<Badge color="primary" textColor="white" size="lg" tagContent="Grande" />
```

### Badge com Ícone no Início

```tsx
<Badge 
    color="primary"
    textColor="white" 
    iconStart={{ iconName: "ArrowLeftIcon", solid: true }}
    tagContent="Com Ícone no Início"
/>
```

### Badge com Ícone no Final

```tsx
<Badge
    color="primary" 
    textColor="white" 
    iconEnd={{ iconName: "ArrowRightIcon", solid: false }} 
    tagContent="Com Ícone no Final" 
/>
```

### Badge com Ícones no Início e no Final
```tsx
<Badge 
    color="primary" 
    textColor="white" 
    iconStart={{ iconName: "ArrowLeftIcon", solid: true }} 
    iconEnd={{ iconName: "ArrowRightIcon", solid: false }} 
    tagContent="Com Ícones" 
/>
```
