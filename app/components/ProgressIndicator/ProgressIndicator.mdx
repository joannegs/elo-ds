
# Progress Indicator

O componente **ProgressIndicator** é utilizado para exibir o progresso de uma ação de forma visual. Ele aceita uma porcentagem para determinar o preenchimento da barra de progresso e suporta customização através de classes adicionais.

----------

## Propriedades (Props)

O componente **ProgressIndicator** aceita as seguintes propriedades:

### percentage

-   **Tipo:** `number` (obrigatório)
-   **Descrição:** Define o progresso atual em porcentagem (de 0 a 100).
-   **Padrão:** Não possui valor padrão, deve ser fornecido.

### className

-   **Tipo:** `string` (opcional)
-   **Descrição:** Permite adicionar classes CSS personalizadas para estilização adicional.

### Atributos HTML

-   O componente aceita todos os atributos padrão de um elemento `<div>`, incluindo:
    -   `id`: Identificador único.
    -   `aria-label`: Para acessibilidade.

----------

## Exemplo de Uso

### Progresso Básico

```tsx
<EloProgressIndicator percentage={50} className="theme-red"/>
```
