
# Button

O componente **Button** é utilizado para criar interações acionáveis na interface do usuário. Ele é estilizado e configurável para atender diferentes cenários e ações, seguindo as diretrizes do design system.

---

## Propriedades (Props)

O componente **Button** aceita as seguintes propriedades:

### `children`

- **Tipo:** `React.ReactNode`  
- **Descrição:** O conteúdo do botão, geralmente texto ou ícones.

### `variant`

- **Tipo:** `"primary" | "secondary" | "tertiary" | "error" | "success" | "alert"`  
- **Descrição:** Define a aparência do botão.  
- **Padrão:** `"primary"`

### `iconStart`

- **Tipo:** `{ iconName: IconName; solid?: boolean }` (opcional)  
- **Descrição:** Ícone exibido antes do conteúdo do botão.

### `iconEnd`

- **Tipo:** `{ iconName: IconName; solid?: boolean }` (opcional)  
- **Descrição:** Ícone exibido após o conteúdo do botão.

### `className`

- **Tipo:** `string` (opcional)  
- **Descrição:** Permite adicionar classes CSS personalizadas para ajustar o estilo do botão.

### `disabled`

- **Tipo:** `boolean` (opcional)  
- **Descrição:** Define se o botão está desabilitado.  
- **Padrão:** `false`

### Atributos HTML

O botão aceita todas as propriedades padrão de um elemento `<button>` no HTML, incluindo:

- `onClick`: Função para manipular cliques.  
- `type`: Define o tipo do botão (`submit`, `button`, ou `reset`).  
- `id`: Identificador único.

---

## Estilização

O botão utiliza estilos padrão definidos por classes Tailwind CSS, ajustados pela propriedade `variant` e pelo estado `disabled`:

### **Classes por `variant`:**

- **Primary:**  
  - Ativo: `bg-primary text-white`  
  - Desabilitado: `bg-disabled text-disabled`

- **Secondary:**  
  - Ativo: `bg-primary-50 text-primary`  
  - Desabilitado: `bg-disabled text-disabled`

- **Tertiary:**  
  - Ativo: `text-primary`  
  - Desabilitado: `text-disabled`

- **Error / Success / Alert:**  
  - Customizados com cores específicas: `bg-error`, `bg-success`, `bg-alert`, etc.

OBS.: Quando a propriedade `variant` não é especificada, o botão vai assumir as cores passadas nas propriedades `customBackgroundColor` e `customTextColor`.

## Exemplo de Uso

### Botão Primário

```tsx
<EloButton variant="primary" onClick={() => console.log('Clicado')}>
  Enviar
</EloButton>
```

### Com Ícone Inicial

```tsx
<EloButton iconStart={{ iconName: 'ArrowLeftIcon' }}>
  Voltar
</EloButton>
```

### Com Ícone Final

```tsx
<EloButton iconEnd={{ iconName: 'ArrowRightIcon' }}>
  Próximo
</EloButton>
```

### Com Ícones em Ambas as Extremidades

```tsx
<EloButton iconStart={{ iconName: 'ArrowLeftIcon' }} iconEnd={{ iconName: 'ArrowRightIcon' }}>
  Ir
</EloButton>
```

### Botão Desabilitado

```tsx
<EloButton variant="primary" disabled>
  Enviar (Desabilitado)
</EloButton>
```
