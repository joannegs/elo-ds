# Header

O componente **Header** fornece um cabeçalho personalizável, permitindo a inclusão de um logotipo e uma navegação.

## Propriedades (Props)

### `logo`
- **Tipo:** `React.ReactNode`
- **Descrição:** Componente ou elemento que representa o logotipo no cabeçalho.

### `navigation`
- **Tipo:** `React.ReactNode`
- **Descrição:** Elementos de navegação exibidos no lado direito do cabeçalho.

## Exemplo de Uso

```tsx
<EloHeader 
  logo="Logo" 
  navigation={
    <div className="flex gap-4">
      <a href="#">Home</a>
      <a href="#">Sistema</a>
    </div>
  }
/>
```

### Exemplo Alternativo

```tsx
<EloHeader 
  logo="Logo"

      <a href="#">Área 1</a>
